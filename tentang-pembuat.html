Tentang Pembuat

Muhammad Fachri Al Husein

Seorang pengguna transportasi umum (terutama Transjakarta), membuat website ini agar berguna mengetahui halte di beberapa koridor, terutama bus stop agar tidak nyasar.

Transportasi Umum Transjakarta Developer

Motivasi

Sebagai pengguna aktif transportasi umum Jakarta, saya sering mengalami kesulitan dalam mencari informasi halte dan rute Transjakarta. Website ini dibuat untuk membantu sesama pengguna transportasi umum agar tidak tersesat dan dapat dengan mudah menemukan halte terdekat.

Visi

Menciptakan platform informasi transportasi yang mudah diakses, akurat, dan bermanfaat bagi masyarakat Jakarta. Website ini diharapkan dapat menjadi referensi utama untuk informasi transportasi umum di Jakarta.

Teknologi yang Digunakan

Frontend Development

HTML5, CSS3, JavaScript, dan Bootstrap 5 untuk membangun antarmuka yang modern dan responsif.

HTML5 CSS3 JavaScript Bootstrap 5

Interactive Maps

Leaflet.js untuk menampilkan peta interaktif dengan performa tinggi dan fitur yang lengkap.

Leaflet.js OpenStreetMap Interactive

GTFS Data

General Transit Feed Specification untuk data transportasi yang standar dan terstruktur.

GTFS Open Data Transit

Icon System

Iconify untuk icon yang konsisten dan mudah dikelola dengan library icon yang lengkap.

Iconify Vector Icons Consistent

Typography

Plus Jakarta Sans untuk tipografi yang modern dan mudah dibaca dengan berbagai variasi.

Plus Jakarta Sans Modern Readable

Responsive Design

CSS Grid, Flexbox, dan media queries untuk layout yang adaptif di semua perangkat.

CSS Grid Flexbox Mobile First
