Kebijakan Penggunaan

Tentang JakMove

JakMove adalah platform informasi transportasi Jakarta yang bersifat open source dan dapat digunakan secara bebas oleh siapa saja. Website ini dikembangkan untuk kepentingan publik dan menggunakan data GTFS (General Transit Feed Specification) yang bersifat publik.

Penggunaan Data

JakMove tidak mengumpulkan data pribadi pengguna. Website ini hanya menggunakan data GTFS yang bersifat publik untuk menampilkan informasi transportasi Jakarta seperti rute, halte, dan jadwal. Tidak ada data pribadi yang disimpan atau diproses.

Penggunaan yang Diizinkan

Anda diperbolehkan untuk:

  • Menggunakan website untuk mendapatkan informasi transportasi Jakarta
  • Menggunakan data GTFS yang tersedia untuk tujuan pendidikan
  • Berkontribusi pada pengembangan proyek melalui GitHub
  • Menggunakan website untuk tujuan pribadi atau non-komersial
  • Mengadaptasi atau memodifikasi kode untuk keperluan pribadi

Penggunaan yang Dilarang

Anda tidak diperbolehkan untuk:

  • Menggunakan website untuk tujuan komersial tanpa izin
  • Menghapus atau mengubah atribusi sumber (JakMove)
  • Menyalahgunakan data untuk tujuan yang merugikan
  • Melakukan reverse engineering atau hacking website
  • Menggunakan konten tanpa mencantumkan sumber asli

Atribusi Sumber (Wajib)

Jika Anda menggunakan atau mengadaptasi konten dari JakMove, Anda wajib mencantumkan:

  • Nama Proyek: "JakMove"
  • Link Website: jakmove.com
  • Pembuat: "Muhammad Fachri"
  • Lisensi: Open Source

Kontribusi

JakMove adalah proyek open source yang mengundang kontribusi dari siapa saja. Dengan berkontribusi, Anda menyetujui bahwa:

  • Kontribusi Anda akan menjadi bagian dari proyek open source
  • Kontribusi akan dilisensikan di bawah lisensi yang sama dengan proyek
  • Anda memiliki hak untuk memberikan kontribusi tersebut
  • Setiap kontribusi akan dipertimbangkan dan dihargai

Pembatasan Tanggung Jawab

JakMove disediakan "sebagaimana adanya" tanpa jaminan apapun. Kami tidak bertanggung jawab atas kerugian yang timbul dari penggunaan website ini. Data transportasi yang ditampilkan dapat berubah dan tidak selalu 100% akurat.

Perubahan Kebijakan

Kami berhak untuk mengubah kebijakan penggunaan ini kapan saja. Perubahan akan diumumkan di website ini dan akan berlaku sejak tanggal publikasi.
